Workflowy MCP

Remote self-hosted MCP access to Workflowy through Vercel, Neon, and Streamable HTTP.

Endpoint

Use this deployment's MCP endpoint in clients that support Streamable HTTP.

https://YOUR-VERCEL-APP.vercel.app/api/mcp

Environment

Add these variables in Vercel before deploying.

DATABASE_URL=postgres://...
ACCESS_SECRET=replace-with-openssl-rand-hex-32

Authorization

Send both your deployment secret and Workflowy API key in the Authorization header. 

Authorization: Bearer ACCESS_SECRET:WORKFLOWY_API_KEY

Claude Code

{
  "mcpServers": {
    "workflowy": {
      "type": "streamable-http",
      "url": "https://YOUR-VERCEL-APP.vercel.app/api/mcp",
      "headers": {
        "Authorization": "Bearer ACCESS_SECRET:WORKFLOWY_API_KEY"
      }
    }
  }
}

Tools

  • list_bookmarks, save_bookmark, delete_bookmark
  • read_doc and edit_doc through Workflowy's LLM Doc API
  • get_targets for inbox and home discovery
